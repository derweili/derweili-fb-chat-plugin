== Description ==

This plugin implements the Facebook Chat with an Opt-In Function to comply with German law.
https://developers.facebook.com/docs/messenger-platform/discovery/customer-chat-plugin



== Installation ==

1. Upload `derweili-fb-chat-plugin.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Customizer and add your Facebook APP ID and your Facebook Page ID
